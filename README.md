# Rocketseat

A Rocketseat é uma plataforma de educação em tecnologia onde você encontra tudo o que precisa para progredir na programação e alcançar seus objetivos no menor tempo possível.

Para isso desenvolvemos uma metodologia de aprendizado orientada à prática, programas de educação de acordo com seu momento na jornada, e uma comunidade incrível com milhares de devs colaborando entre si.

Temos o Discover para iniciantes com aulas gratuitas, o programa de aceleração Ignite para quem já domina os fundamentos da programação avançar na carreira, e o Experts Club para você manter o aprendizado contínuo.

A nossa plataforma une toda a jornada para você encontrar tudo que precisa em um único lugar. Portanto as empresas também fazem parte do ecossistema e se conectam aos devs através da plataforma. Preparar ótimos devs e uni-los a ótimas empresas: essa é a nossa busca diária.

O aprendizado é contínuo e a melhor forma de evoluir é juntos 🚀
